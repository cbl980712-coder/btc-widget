=====================================
  BTC 合约判断面板 - 使用说明
=====================================

【首次使用】
1. 确保电脑已安装 Python 3.x
   下载地址：https://www.python.org/downloads/
   安装时勾选 "Add Python to PATH"

2. 双击 start.bat 启动

3. 浏览器会自动打开面板

【功能说明】
- 支持 BTC/ETH/SOL/BNB/DOGE/XRP
- 数据来自 Binance 公开 API，无需 Key
- 每30秒自动刷新
- 显示：趋势方向、持仓结构、资金费率、
        进场/止损/止盈参考、判断依据

【切换品种】
点击顶部按钮即可切换

【停止服务】
在 start.bat 窗口按任意键停止

【注意事项】
- 仅供参考，不构成投资建议
- 网络需能访问 fapi.binance.com
